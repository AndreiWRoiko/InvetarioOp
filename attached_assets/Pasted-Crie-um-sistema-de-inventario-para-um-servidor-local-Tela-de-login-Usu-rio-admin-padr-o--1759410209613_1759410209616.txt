Crie um sistema de inventario para um servidor local.
  Tela de login
    Usuário admin padrão
    Parte de cadastro e autorização de novos usuários, com um painel de gestão.
      Validar e direcionar a uma das opções de visualização
        Admin(Usar tudo como o usuário padrão)
        Suporte(fazer o controle do inventario e modificação dos itens)
        Controle(Faz a verificação de dashboards e somente os inventarios)
